Data Source

This dashboard reads a single file, data.json, served next to this page.

📊 data.json is rebuilt on a schedule by a GitHub Action (scripts/fetch_data.py) that pulls CPI & PPI from BLS and the Fed Funds rate from FRED. BLS and FRED keys live as encrypted GitHub repo secrets — never in your browser. The page makes no cross-origin API calls, so there is no CORS dependency.
How it refreshes

The Action runs automatically (and can be triggered manually from the repo's Actions tab). When BLS or FRED publishes new figures, the next run updates data.json and the dashboard picks it up on reload — click Refresh Data any time to re-read the latest file.

⚠ If a scheduled pull fails, the dashboard shows Data Pull Failed (E###) with a numbered code instead of estimated or stale figures. Codes are documented in the repo README.
